IL CMD DI WINDOWS

In informatica, il prompt dei comandi o cmd.exe è una shell a riga di comando dei sistemi operativi Windows NT. È un'applicazione console che consente agli utenti di interagire con il computer tramite un'interfaccia a riga di comando

il cmd può essere usato sia lanciando il programma normamente o aprendolo con i permessi avanzati ("amministratore")

differenze tra cmd uso normale e cmd permessi d' amministratore

  • con i permessi normali si può: creare cartelle distruggere cartelle ecc..
  • con i permessi d' amministratore si può:fare modifiche al sistema..

  • ATTENZIONE!

    un uso sbagliato di commai con permessi avanzati può portare alla perdita di dati o la cancellazione della cartella windows32
